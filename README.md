# OPIcPractice
2021.12.22에 제작한 OPIC 연습용 프로그램 소스코드입니다.
---
**가장 중요한 Excel의 내용은 공개 및 공유하지 않습니다**

----
#### 사용한 라이브러리
openpyxl, PyQt5, gtts, pyaudio, playsound

---
#### 사용을 위한 제한 조건
질문 리스트가 정리 및 저장되어 있는 excel이 필요합니다. (공유 절대 안 합니다)

정품 인증 된 Excel 프로그램이 필요합니다.

마이크가 연결되어 있어야 합니다.
